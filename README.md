## Core Research Theme
This series of papers establishes a rigorous **solution complexity k(n) / verification complexity z(n)** framework to:
- Formalize the k/z model and eliminate implicit assumptions in naive P vs NP arguments
- Dissect logical flaws in intuitive P≠NP proofs (upper/lower bound confusion, circular reasoning, precondition inconsistency)
- Translate the three classic complexity barriers into the k/z framework
- Prove the **conditional equivalence principle**:  
  *Under fixed standard computational preconditions, P vs NP remains open; under changed models, both P=NP and P≠NP can be rigorously established*
- Characterize valid boundaries of the k/z intuition in restricted computational models
